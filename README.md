  # 🎨 SocialCanvas
  
  **Generator Komentar Sosial Media Lengkap & Realistis!**
  
  Buat tangkapan layar komentar secara *live* untuk kebutuhan konten kreatif, edukasi, atau meme lucu. Mendukung berbagai platform populer dengan kostumisasi yang sangat mendetail.

## ✨ Fitur Utama

- 📱 **Beragam Platform:** Menirukan antarmuka komentar dari **TikTok**, **Instagram**, **YouTube**, **X (Twitter)**, dan **Kick**.
- 🎨 **Kustomisasi Penuh:** Kamu berkuasa penuh untuk mengubah *Avatar*, *Username*, *Isi komentar*, *Jumlah Like*, hingga *Waktu posting*.
- 💬 **Balasan & Sub-Komentar:** Mendukung style 'Reply Bubble' yang unik dan penumpukan komentar (*stacked comments*).
- 🎲 **Serba Otomatis (Randomizer):** Ganti profil dan isi komentar secara acak (*Acak Profil* & *Randomize All*) dengan satu tombol saja! Ditenagai oleh [DiceBear](https://www.dicebear.com/).
- 🌙 **Dark/Light Mode:** Sesuaikan tampilan pratinjau (preview) hasil komentar sesuai keinginan.
- 🖼️ **Ekspor Beresolusi Tinggi:** Simpan komentarmu menjadi PNG bersih dan bebas repot (dilengkapi opsi *Background Transparan*).
- ✒️ **Akurasi Font:** Sesuaikan *Font Style* seperti **Roboto (Android)** atau **San Francisco (iOS)** agar makin realistis!

## 🚀 Cara Menjalankan

Ikuti panduan singkat ini untuk melakukan *development* secara lokal:

### 1. Install Dependensi
Pastikan kamu telah menginstal Node.js versi terbaru, lalu jalankan perintah berikut di direktori proyek:
```bash
npm install
```

### 2. Mulai *Dev Server*
```bash
npm run dev
```

### 3. *Build* ke Resolusi Produksi
```bash
npm run build
```

## 🛠️ Teknologi yang Digunakan

Aplikasi **SocialCanvas** dikembangkan dengan stack *frontend* yang *modern* dan cepat:
- **Platform:** React dengan TypeScript untuk komponen *type-safe*.
- **Styling UI:** Tailwind CSS utuk styling cepat dan responsif serta ikon-ikon elegan dari Lucide React.
- **Build Tool:** Vite.
- **Penyimpanan Gambar:** Menggunakan pendekatan `html2canvas` dan HTML native snapshot.
